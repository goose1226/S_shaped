# My Python Project

## 安裝步驟
1. 克隆專案：
    ```bash
    git clone https://github.com/yourusername/your-repository.git
    ```
2. 安裝依賴：
    ```bash
    pip install -r requirements.txt
    ```

## 使用說明
- 啟動應用：
    ```bash
    python play5.py
    ```

- 用途：
    這個專案控制一個車子，並通過 Flask 查看 藉由車道辨識自動前進、後退、轉彎等動作，通過S型車道線。
